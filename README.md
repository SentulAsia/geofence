Geofence implementation sample.

## Usage

### Prerequisites

* iOS 11.0+
* Xcode 10.3+

### Installing

1. Clone this repository.
2. Change the bundle identifier, team cert and provisioning profile.
3. CMD + U to make sure the unit test pass.
4. Build and run.
